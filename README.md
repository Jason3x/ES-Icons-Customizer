ES Icons Customizer is a Bash script for dArkOS RE and darkos4clone that lets you replace the standard EmulationStation WiFi / Bluetooth status icons with cute original designs (Kirby, Ghost, Totor8o, Alien, …) and fully customize their colors.

No more boring network icons – give your R36S a personality!

## ✨ Features
- 9 icon themes for both WiFi & Bluetooth:
Kirby · Space Invader · Ghost · Totoro · Robot · Alien · Skull · Cat

- Live preview – see the icon before applying (requires optional deps)

- Persistent color settings – your chosen colours are saved and reused

- One‑click restore – original icons are backed up automatically on first run

## 📋 Requirements
R36S device running dArkOS RE or darkos4clone

Internet connection (only needed for first‑time preview dependency install)

## 🚀 Installation
Download ES_Icons_Customizer.sh

Copy it to: roms/tools

Launch it from the Tools section on your device

The script will create a backup of your original icons the first time you run it.

## 🕹️ Usage
From the main menu you can:

- Configure Colors	Change colours for each icon state (active, idle, off, share, service)
- Change WiFi icon only	Pick a theme just for WiFi
- Change Bluetooth icon only	Pick a theme just for Bluetooth
- Change both icons	Choose a theme for WiFi and one for Bluetooth, apply both at once
- Restore original icons	Revert to the factory icons (backup required)
- Install Preview Dependencies
  
After applying new icons, EmulationStation restarts automatically – the changes appear immediately.

## 🔧 Dependencies (optional but recommended for preview)
- converts SVG to Png.
- displays images in the terminal
